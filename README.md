## About Agent Sessions Day

Agent Sessions Day is a global livestream event hosted by Microsoft and GitHub, showcasing the future of AI-powered development with Visual Studio Code. This event highlights how developers can work with multiple AI coding agents in a unified experience, including GitHub Copilot, and demonstrates the latest advancements in agentic development workflows.

## Sweepstakes Overview

Participate in the Agent Sessions Day sweepstakes for a chance to win exciting prizes! This sweepstakes is designed to engage the developer community and celebrate the evolution of AI-assisted coding.

### Eligibility

- Open to all registered attendees of Agent Sessions Day
- Must be 18 years or older
- Void where prohibited by law
- See official rules for complete eligibility requirements

### How to Participate

1. **Register**: Sign up for Agent Sessions Day on the [Microsoft Reactor](https://developer.microsoft.com/en-us/reactor/) event page
2. **Attend**: Join the livestream event
3. **Engage**: Participate in demos, Q&A sessions, and community activities
4. **Enter**: Follow the instructions provided during the event to enter the sweepstakes
